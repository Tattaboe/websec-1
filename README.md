# Безопасность веб-приложений. Лабораторка №1

## Структура проекта

- `index.html` – основная страница с разметкой калькулятора.
- `style.css` – таблица стилей, отвечающая за внешний вид.
- `script.js` – логика работы приложения.

### index.html

- Зелёный контейнер (`greenBox`) с полями ввода чисел (`firstNum`, `secondNum`), выпадающим списком операций (`operator`) и кнопкой «подсчитать» (`calcButton`).
- Поле вывода (`outputBox`) разделено на две части: для предыдущего результата (`oldResult`) и текущего (`newResult`).
- Блок для всплывающих сообщений об ошибке (`warningMessage`).

### style.css

- Стилизация элементов в точности, как на поясняющем рисунке.
- Компоновка страницы адаптирируется под размер экрана.

### script.js

- **`showErr`** – отображает сообщение об ошибке возле указанного поля, подсвечивает его и скрывает сообщение через 3 секунды.
- **`clearErr`** – убирает подсветку ошибок и скрывает всплывающее окно.
- **`getNums`** – проверяет поля на пустоту и корректность ввода (числа), возвращает объект с числами или `null` при ошибке.
- **`calc`** – выполняет выбранную арифметическую операцию на основе переданных чисел и оператора.
